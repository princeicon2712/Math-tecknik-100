## সরল অংক - BODMAS

BODMAS মানে:

B = Brackets (বন্ধনী)

O = Orders (ঘাত, √ ইত্যাদি)

D = Division (÷)

M = Multiplication (×)

A = Addition (+)

S = Subtraction (−)


D(Divition) এবং M (Maltiaplicatiom) — এরা সমান স্তরের অপারেশন, মানে কারও অগ্রাধিকার নেই।

✅ তাই আমরা এই দুইটি অপারেশন বাম থেকে ডানে করি।
